# 📱 SocialSnap

A Full Stack Social Media Web Application built using React, Node.js, Express.js, MongoDB, Cloudinary and JWT Authentication.

---

## 🚀 Features

- 👤 User Registration & Login
- 🔐 JWT Authentication
- 🚪 Logout
- 📝 Create Post
- 🖼️ Upload Images
- 🎥 Upload Videos
- 🎵 Upload Audio
- 👤 User Profile
- ✏️ Update Profile
- 📷 Upload Profile Image
- ❤️ Follow / Unfollow Users
- 📱 Responsive Frontend

---

## 🛠️ Tech Stack

### Frontend
- React
- React Router DOM
- Axios
- CSS

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- Multer
- Cloudinary
- Cookie Parser
- Bcrypt

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/vijayvar440/SocalSnap.git
```

### Backend

```bash
cd BACKEND
npm install
npm run dev
```

### Frontend

```bash
cd Frontend
npm install
npm run dev
```
